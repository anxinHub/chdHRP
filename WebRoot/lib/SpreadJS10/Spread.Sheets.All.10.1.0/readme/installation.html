Instructions

Use the following steps to install Spread.Sheets.

To install Spread.Sheets,

  1. Save the downloaded ZIP file (Spread.Sheetsxxxx.zip) to a temporary directory on your system, and then unzip the files to a directory.

The default Spread.Sheets folders (Windows only):

Install Directory File Description
\Spread.Sheets.xxxx   Main directory for product
\Spread.Sheets.xxxx\css   CSS files
\Spread.Sheets.xxxx\samples   Sample pages for the widget
\Spread.Sheets.xxxx\scripts gc.spread.sheets.all.xxxx.min Minimized js file for widget which includes all functions
\Spread.Sheets.xxxx\definition GC.spread.sheets.d.ts TypeScript API definition file
\Spread.Sheets.xxxx\scripts\pluggable   Pluggable js files
\Spread.Sheets.xxxx\scripts\interop angular.gcspread.sheets.xxxxx.min AngularJS support files
  gc.spread.excelio.xxxxx.min Client-side Excel import and export support files
  gc.spread.sheets.migration.xxxxx.min Migration file for backward compatibility
\Spread.Sheets.xxxx\scripts\resources ja\gc.spread.sheets.resources.ja.xxxxx.min
zh\gc.spread.sheets.resources.zh.xxxx.min		 Japanese and Chinese resource files

 

To install the Spread.Sheets designer,

  1. Save the downloaded ZIP file (SpreadSheetsDesigner.zip) to a temporary directory on your system, and then unzip the files to a directory.
  2. Run the setup file for your environment.
  3. Run the designer, select the lock icon, and enter your license key to unlock the Spread.Sheets Designer.

Note: The evaluation setup is for a trial and the production setup is for a licensed version of the product.

Platform Setup
Windows Spread.Sheets-Designer-Evaluation-Setup.10.1.0.exe or Spread.Sheets-Designer-Production-Setup.10.1.0.exe
Mac Spread.Sheets-Designer-Evaluation-10.1.0.dmg or Spread.Sheets-Designer-Production-10.1.0.dmg
Linux Spread.Sheets-Designer-Evaluation-10.1.0-x86_64.AppImage or Spread.Sheets-Designer-Production-10.1.0-x86_64.AppImage

The default Spread.Sheets designer folder:

Install Directory Description
C:\Program Files (x86)\Spread.Sheets Designer\10.1.0 Designer directory for product

If you downloaded the Excel Import and Export Component, refer to Excel Component Installation for installation information.

Refer to Trial Version and Licensing Information for licensing information.
