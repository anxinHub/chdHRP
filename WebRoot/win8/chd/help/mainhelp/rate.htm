第一章 数据采集

1.1 概述

本模块完成对项目月收入数据的采集和维护，每月从HIS系统导入项目收入月结汇总表或直接通过手工录入并保存到数据库中。

1.2 数据导入

功能

将HIS系统中的项目收入数据导入CBCS的数据库中并可实现删除。根据用户定义的import文件执行数据导入。

 

页面元素说明

项目

含义

核算月

通过下拉列表选择数据导入的月份

报表

通过下拉列表选择导入的数据类型

清除本月数据

通过下拉列表选择导入数据时是否清除对应月份的数据

强制导入

通过下拉列表选择是否进行数据的强制导入，当选择“是”时，执行［导入］时经数据检查发现非法数据时只导入正确数据，非法数据不导入并对本次操作结果进行提示；当选择“否”时，执行［导入］时经数据检查发现非法数据时中止［导入］操作并对本次操作结果进行提示。

［检查］

点此按钮执行数据检查，并对检查结果进行提示

［导入］

点此按钮执行对应选定核算月的数据导入

［删除］

点此按钮执行选定核算月中已导入的数据

 

提示

1、操作者根据用户实际修改C:\cbcsV2.0.0\webapps\ROOT\WEB-INF\import.xml。的Excel表必须严格按照收入数据表的格式才允许导入。

2、修改import.xml文件后，需重启CBCS服务修改方能生效。

第五章 单位成本计算

5.1 单位成本计算

功能

按照参数设置计算单位成本。

操作如下：

选择菜单“单位成本计算”à“单位成本核算”，进入如下页面

选择配置编码点击［计算］，系统按选定的配置编码下的时间范围作单位成本作计算，输入文件编号并计算结果保存到该文件下，选择［计算］操作完成后系统提示“单位成本计算成功”。

提示

²        cbcsV2.0.0的项目核算在保留收支配比法的基础上增加单位成本计算方法。单位成本计算方法是由医疗技术部门的专家及财务、物价等人员对成本项目制定单位成本比例。根据制定好的单位成本比例，计算各医疗项目的比价关系，即分配的比例，并据此比例计算出医疗项目成本。用户可以根据实际需要选择具体的核算方法。

²        具体设置方法为修改cbcsV2_0\webapps\ROOT\WEB-INF\congfig.xml文件中配置

若选择人工作业成本、直接成本单独计算，则修改：

<!--成本当量计算/人工作业成本是否单独计算 true 是 false 否-->

<is_ec_acomp>true</is_ec_acomp>

<!--成本当量计算/直接成本是否单独采集 true 是 false 否 -->

<is_cdirect>true</is_cdirect>；

否则，选择false，注意修改设置后须重新启动CBCS服务，修改方能生效。

操作如下：

删除参与单位成本计算科室记录

如果用户需要废弃某些错误的记录，或对某条记录不作修改而想从系统直接删除它，则在查询页面，在选择框中对该条记录“打钩”选中，然后点击［删除］，系统提示“是否删除”，点击［确定］，执行删除；点击［取消］，放弃删除。

提示
