医院成本管理系统

（内部分配）

用 户 手 册

第一章 分配定义

1.1 概述

本模块是本系统的基础。实现定义参与计算的奖金项目、收入比例、成本项目、考核指标及科室奖金分配方案。

1.2 奖金项目

功能

列表显示系统预置和由用户自定义的参与科室奖金分配和计算的奖金项目，并可对自定义的奖金项目进行维护，包括添加、停用、启用等。

页面元素说明

［全选］

选定所有查询记录

［重置］

取消最近修改的所有录入信息

［停用］

停用选定的奖金项目

［启用］

启用选定的奖金项目

［添加］

添加病人收入数据

添加奖金项目

操作如下：

在页面点击［添加］，进入奖金项目添加页面

通过下拉列表选择“奖金项目类别”、“医疗项目类”，依次输入“奖金项目名称”、“公式”、“说明”，点击［添加］，提示“增加成功”，本条记录添加完成。重复以上操作，直至所有的奖金项目添加完毕。点击［返回］回至上级奖金项目查询页面。

停/启用奖金项目

操作如下：

1、选择欲停用的奖金项目，在奖金项目查询页面将“项目名称”前对应的“选择”框选中，点击［停用］，若确实停用某奖金项目，点击［确定］，该项目奖金停用，不参与科室奖金的计算和分配；否则点击［取消］，该奖金项目不停用，仍参与奖金分配。

2、对于已停用的奖金项目，在奖金项目查询页面将“项目名称”前对应的“选择”框选中，点击［启用］，若确实启用某奖金项目，点击［确定］，该项目奖金重新启用；否则点击［取消］，取消启用。

提示                                                                             

²        奖金项目里的“效益奖金”、“平均奖金”、“服务奖金”及公式为系统预置项，只能查看，不能删改或停用。

²        本系统的科室类别分3类：经营科室、管理科室、服务科室。对应的奖金构成分别为：

经营科室奖金=效益奖金+项目奖金；效益奖金公式如奖金项目查询页面定义  

管理科室奖金=平均奖金；平均奖金公式如奖金项目查询页面定义

服务科室奖金=服务奖金；服务奖金公式如奖金项目查询页面定义

²        用户自定义的奖金项目仅作为经营科室的项目奖金项并参与计算，不参与管理/服务科室的奖金分配或计算。

1.5 考核指标

功能

列表显示已定义的科室考核指标项目，并可对考核指标进行修改、删除、添加操作。

添加考核指标

操作如下：

在页面点击［添加］，进入考核指标添加页面，依次输入“考核指标名称”、“考核指标公式”，选择“考核指标类别”，输入完毕点击［添加］，页面提示“添加成功”，点击［返回］至上级考核指标查询页面。

停/启用考核指标

操作如下：

1、在考核指标查询页面将“指标编码”前对应的“选择”框选中，点击［停用］，若确实停用某奖金项目，点击［确定］，该考核指标停用，不参与奖金的计算；否则点击［取消］，该奖金项目不停用，仍参与奖金分配。

2、对于已停用的考核指标，在考核指标查询页面将已停用的考核指标对应的“选择”框选中，点击［启用］，点击［确定］，该考核指标重新启用；否则点击［取消］，取消启用。

添加科室奖金分配方案

操作如下：

1、在科室奖金分配方案查询页面选择分配科室，依次点击［配置］、［查询］，进入奖金分配方案添加页面：

2、在页面将“奖金项目编码”前对应的“选择”框选中，依次定义比例系数和成本调控金额，然后点击［保存］，提示“保存成功”，点击［返回］至上级奖金分配方案查询页面。

3、重复以上步骤，直至所有的奖金分配方案定义完毕。

提示

²        定义/添加科室奖金分配方案时，奖金项目必须选择“效益奖金”、“平均奖金”、“服务奖金”之一。

²        项目奖金仅能和“效益奖金”同时选择，不能和“平均奖金”、“服务奖金”共存。

修改科室奖金分配方案

操作如下：

在科室奖金分配方案查询页面点击“分配科室编码”链接，进入科室奖金分配方案修改页面，修改相关信息后，点击［保存］，提示“保存成功”，保存修改后的结果，点击［返回］至上级科室奖金分配方案查询页面。

删除科室奖金分配方案

操作如下：

如果需要删除某些已定义的奖金分配方案，或对某些分配方案不作修改而想直接删除，则在科室奖金分配方案查询页面，在选择框中对该条记录选中，点击［删除］，系统提示“是否删除”，点击［确定］，执行删除；点击［取消］，放弃删除。

第二章 科室奖金分配

2.1 概述

本模块实现对已完成科室核算的各类科室奖金的分配和查询。包括经营科室的分配和查询、管理科室的分配和查询、服务科室的分配和查询及按年度查询各类科室奖金的分配状态。

2.2 控制定额

2.2.1 定额维护

功能

对指定年份的全院奖金定额进行维护。包括查询、修改。

操作如下：

1、在奖金定额维护页面选择期间，点击［查询］，页面下方刷新显示对应期间的“奖金定额维护表”。

2、对于未完成奖金分配的月份，修改月度奖金定额后，点击［保存］，提示”保存成功“。

提示

²        对于已完成奖金分配的月份，奖金定额只能查看不能修改；对于未完成奖金分配月份的奖金定额，允许修改和查看。

²        全院奖金定额=SUM（经营科室定额+管理科室定额+服务科室定额），由系统自动计算。

2.2.2 执行查询

功能

查看指定年份的全院奖金定额和实际金额。

提示

²        对于实际金额高于定额的月份，实际金额数值以红色显示。

2.3 经营科室

功能

实现对指定核算月的经营科室奖金分配，并列表显示分配结果。

页面元素说明

项目

含义

核算月

通过下拉列表选择经营科室奖金分配的核算月

［查询］

点此按钮，查询选定分配核算月的经营科室奖金分配结果

［初始化］

点此按钮，根据系统设置进行经营科室奖金分配的初始化判断和处理工作

［打印］

点击按钮，打印当前经营科室奖金分配结果表

 

提示

²        选择的核算月必须已经完成了科室核算方可进行奖金分配。

²        科室奖金分配状态分4种：

未完成科室核算，不能进行奖金分配；

已完成科室核算，可以进行奖金分配；

奖金分配进行中；

奖金分配完成，不能修改，只能查看；

²        可以对已经完成奖金分配或正在进行分配的经营科室的奖金分配情况进行查询。

²        直接收入、成本、加减项1、考核分数、项目奖金、乘除项、加减项2提供链接，可查看明细。

2.4 管理科室

功能

实现对指定核算月的管理科室奖金分配，并列表显示分配结果。

操作同5.3“经营科室”，略。

提示

²        选择的核算月必须已经完成科室核算且在经营科室奖金分配完成的基础上进行分配。

²        可以对已经完成奖金分配或正在进行分配的管理科室的奖金分配情况进行查询。

²        考核分数、成本金额、乘除项、加减项2提供链接，可查看明细。

2.5 服务科室

功能

实现对指定核算月的服务科室奖金分配，并列表显示分配结果。

操作同5.3“经营科室”，略。

提示

²        由于服务科室的奖金分配计算不涉及经营科室和管理科室的奖金分配数据，故服务科室的奖金不必在经营科室和管理科室完成对应核算月奖金分配的基础上进行。

²        可以对已经完成奖金分配或正在进行分配的服务科室的奖金分配情况进行查询。

²        成本、考核分数、乘除项、加减项2提供链接，可查看明细。

查询指定年份的科室奖金分配状态，并可对已完成奖金分配的科室查看分配明细表。

第三章 个人奖金分配

3.1 概述

本模块实现对已完成科室奖金分配的科室进行个人奖金的分配。包括个人奖金分配、查询和查看分配状态等。

3.2 奖金分配

功能

对已完成奖金分配的科室进行个人奖金分配。

操作如下：

1、依次选择“核算月”、“分配科室”、“奖金类别”，点击［查询］、［初始化］，进入个人奖金分配设置页面

2、定义好分配方案后，点击［保存］，保存定义好的分配方案，此时仍可以修改相关输入项的值，使科室总奖金与个人实发奖金之和相等，点击［完成］，个人奖金分配完成，此时不能再修改相应输入项。

3、重复以上操作，直至所有科室的个人奖金分配完成。

4、点击［打印］，打印当前奖金分配结果表。

提示

²        考核成绩提供链接，可查看明细。
